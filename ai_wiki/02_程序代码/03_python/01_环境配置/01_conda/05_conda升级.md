1. 更新conda
    ```bash
    conda upgrade conda
    ```
    如果上一步更新失败，执行这个命令即可
    ```bash
    conda update --force conda
    ```

2. 更新anaconda
    ```bash
    conda update anaconda
    ```

3. 安装升级python
    这个命令默认升级到最新版本
    ```bash
    conda install python
    ```
    如需指定版本
    ```bash
    conda install python=3.8
    ```
